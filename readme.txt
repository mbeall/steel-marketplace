== Description ==

A plugin that is part of the Sparks Framework. Extends Steel and creates an easy ecommerce development framework.

For best results, use the theme (or child theme of) Flint and activate the plugin Steel first.

Like our plugin? Write us a review!

== Installation ==

1. Upload `steel-marketplace.zip` to the `/wp-content/plugins/` directory
1. Unzip `steel-marketplace.zip` to a sub-directory named `steel-marketplace`
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Check out the options menu for more options
